# SmartResume AI

SmartResume AI is a simple resume and job description matcher built with Streamlit and Python.

## Features
- Upload a resume PDF
- Paste a job description
- Extract and preview resume text
- Prepare for AI-based match analysis

## Tech Stack
- Python
- Streamlit
- pdfplumber
- OpenAI API

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
